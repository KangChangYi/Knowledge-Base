# 块级元素与行内元素

这里只挑选了较为常用的元素列出。

# 块级元素：

标签名 语义 特殊
div 文档分区
ul 无序列表元素
p 文本段落
hr 水平线
header 网页的标题或头部 H5
footer 页脚或网页的底部 H5
canvas 被用来绘制图形 H5
form 内含表单
table 二维数据表
aside 页面中独立的部分，如侧边栏 H5
article 页面中可复用的结构，如用户的帖子、评论 H5
section 同 aside 类似，但通常带标题 H5
pre 预定义格式文本 其中的空格和换行符都会显示出来

# 行内元素

标签名 语义
button 按钮
span 最常用的行内元素
script 包含 js 代码
input 表单输入元素
label 为 input 定义标注
a 通向其他 URL 的超链接
img 图片
select 选择器
textarea 多行文本框
br 换行符
code 代码文本
