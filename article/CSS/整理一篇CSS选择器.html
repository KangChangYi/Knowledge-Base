# 整理一篇完整的 CSS 选择器

有时候遇到想用的选择器忘了什么作用，就还得去搜一下，干脆整理一波。

# 常用选择器

选择器
名称
描述
*
 通配选择器 选择所有
tagName
 元素选择器 通过标签名选择
#idName
 id 选择器 通过元素 id 属性值选择
.className
 类选择器 通过元素 class 属性值选择

# 选择器分组

使用 , 分隔选择器：

A , B { color: red }

A 和 B 字色都变红

# 关系选择器

选择器
名称
描述
E N
 包含选择器 选择所有包含在 E 元素里的 N 元素
E > N
 子选择器 选择所有 E 元素的子元素 N
E + N
 相邻兄弟选择器 选择紧接 E 元素的 N 元素
E ~ N
 兄弟选择器 选择相同父元素中 E 元素之后元素 N

注意：

  • 包含选择器会选择符合条件的所有元素，但子选择器只会选择符合条件的子元素
  • 相邻兄弟选择器必须是 E 的下一个元素 B，并且有相同的父元素

# 属性选择器

下面 att 指的是元素的属性，比如 <a> 标签的 href 属性

选择器
描述
E[ att ]
 选择带有指定属性的 E 元素
E[ att = val ]
 选择带有指定属性且值等于 val 的 E 元素
E[ att ~= val ]
 选择属性值中包含 val 的 E 元素
E[ att = val ]
 选择属性值开头是 val 的 E 元素，必须是整个单词
E[ att ^= val ]
 选择属性值开头是 val 的 E 元素
E[ att %= val ]
 选择属性值结尾是 val 的 E 元素
E[ att *= val ]
 选择属性值包含 val 的 E 元素

注意：

  • 😭 markdown 的表格问题导致 | 打不出来第四个属性选择器中的 \ 应该是 |
  • 属性选择器可单独使用，比如：[ att ]
  • 相邻兄弟选择器必须是 E 的下一个元素 B，并且有相同的父元素

# 伪类选择器

选择器
描述
:link
 超链接在未访问前的样式
:visited
 超链接在访问后的样式
:hover
 鼠标悬停在元素时的样式
:active
 元素在激活（鼠标点击与释放之间）时的样式
:focus
 元素在成为输入焦点（onfocus事件发生）时的样式。(一般应用于表单元素)
:checked
 处于选中状态的元素。(用于input type为radio与checkbox时)
:enabled
 处于可用状态的元素。(一般应用于表单元素)
:disabled
 处于禁用状态的元素。(一般应用于表单元素)
:empty
 没有子元素（包括text节点）的元素
:root
 匹配E元素在文档的根元素。在HTML中，根元素永远是HTML
:not(s)
 匹配不含有 s 选择器的元素
:first-child
 匹配元素的第一个子元素
:last-child
 匹配元素的最后一个子元素
:only-child
 匹配元素仅有的一个子元素
:nth-child(n)
 匹配元素的第n个子元素
:nth-last-child(n)
 匹配元素的倒数第n个子元素E
:first-of-type
 匹配同类型中的第一个同级兄弟元素
:last-of-type
 匹配同类型中的最后一个同级兄弟元素
:only-of-type
 匹配同类型中的唯一的一个同级兄弟元素E
:nth-of-type(n)
 匹配同类型中的第n个同级兄弟元素
:nth-last-of-type(n)
 匹配同类型中的倒数第n个同级兄弟元素E

🤮 令人窒息，我感觉有些选择器这辈子都不会用到

关于 :not() 的用法： 假定有个列表，每个列表项都有一条底边线，但是最后一项不需要底边线。

    li:not(:last-child) {
        border-bottom: 1px solid #ddd;
    }

# 伪元素选择器

选择器
描述
:before / ::before
 在目标元素的前面插入的内容。用来和content属性一起使用
:after / ::after
 在目标元素的后面插入的内容。用来和content属性一起使用
:first-letter / ::first-letter
 设置元素内的第一个字符的样式
:first-line / ::first-line
 设置元素内的第一行的样式
::placeholder
 设置元素文字占位符的样式。(一般用于input输入框)
::selection
 设置元素被选择时的字体颜色和背景颜色
