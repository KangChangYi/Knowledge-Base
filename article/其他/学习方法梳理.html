# 学习方法梳理

万千智慧始于记忆 —— 埃斯库罗斯

# 前言

前段时间当当网 520 活动的时候买了一批书，选择先看完这本。

想看这本书的原因很简单，磨刀不误砍柴工，想通过这本书来掌握一些好的学习方法和学习习惯，反观大学四年，前两年没在学习，后两年在理论上的自学也没有特别好的效果，我想原因就是因为没有使用一套正确的学习方法去学习。

看完这本书之后，我收获了很多有效的学习方法，同时也了解了最无效的学习方法，而我们绝大多数人使用的居然就是这种最无效的学习方法😑。

本文将带你了解，有效的以及无效的一些学习方法。

# 学习方法们

猜一下哪几种方法是相对比较差的

想要学以致用，就必须记忆，只有这样，以及学习的内容才不会在将来需要的时候消失。

并且我们需要坚持不懈的去学习和记忆，擅长学习的人会终身收益。

# 集中练习

# 低效🐌

大多数人在学习一门新的技术或知识的时候，总是倾向于反复阅读，然后进行集中练习，也就是机械的、快速的重复一些东西，就像是考试前的“填鸭”。这种学习方法从长久学习的角度来看，是效率最低的一种方式，但是从临时抱佛脚的角度来看，它确实是一种有效的方法。

  • 集中练习是最低效的，就像在沙子上写字，今天写了，过几天就没了。

# 造成错觉😵

反复阅读 和 集中练习会让人对资料更熟悉，阅读更流畅，以为自己以及掌握了新知识，但其实是一种假象，因为这种轻松学习的方法不会产生持久的记忆，只是在浪费时间。

  • 学习越轻松效果越好吗？恰恰相反，研究表明，但学习更吃力的时候，记忆更持久，更有效。

# 研究成果 📑

  • 研究表明：我们刚才还读过或听到的东西，有 70% 左右会在极短的时间内被忘记，在这之后，遗忘的速度开始变慢，剩下的 30% 会被缓慢遗忘。

所以改善学习方法的一大挑战就是想办法中断遗忘的过程，如何中断遗忘过程？

# 检索练习

# 有效🛵

“检索练习” 意味着自我测验，从记忆中检索知识和技能，是一种值得使用的有效的学习方法。

在高中时期，老师每天的小测验，就是一种检索练习，可以有效的巩固知识，但是现在没有老师了，我们自己如何使用检索练习？

举个栗子🥔：

当你正在学习知识的时候，要不时的停下来，问自己：核心概念是什么？哪些概念我没接触过？我怎么定义这些概念？它跟我已知的东西有什么联系和相似之处？

通过这些问题在你的脑海中检索刚才学习的内容，写出自己的答案，然后再去看资料校验是否正确。（注意一定是从脑海中检索，而不是从资料中去检索，不然就成了反复阅读。）

  • 检索付出的努力越大，记忆效果越好

# 有关检索练习的研究成果 📑

  • 研究人员发现，集中练习（填鸭）能让人在即将到来的考试中取得较高分数。对于使用集中练习的人来说，在第一次考试后时隔两天再考一次，他们就已经忘掉了第一次考试时所记住东西的 50%，而同期进行了检索练习的人，遗忘的信息量只占前次考试时的 13%。

  • 研究人员让学生们听一段故事，里面涉及了 60 个实物的名称。学生在首次接触这些实物名称后立即参加测验，之后便可记住首次测验中内容的 53%，一周之后，记住的内容下降到了 39%。另一组学生学习相同的内容，但不参加测验，一周后他们只记得 28% 的内容。仅需一次检测就可以把记忆量提高 11%。

# 间隔练习

# 高效🚗

如果说检索练习是“健身”，那间隔练习就是有时间安排的“健身”。

健身和学习一样，想要达到最佳效果，就必须有安排，重复多次检索，而且检索之间要有间隔。这样才能让人努力达成认知，而不是无意识的背诵。
